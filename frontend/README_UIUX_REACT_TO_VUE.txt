FINSTART UI/UX REACT -> VUE

This frontend is converted from the supplied React design into Vue SFC files.
- UI component files: src/App.vue and src/components/*.vue
- There are no React dependencies and no .tsx/.jsx files in src.
- Backend, SQL, routes, and database are not included or changed by this frontend package.

Run:
  npm install
  npm run build
  npm run dev

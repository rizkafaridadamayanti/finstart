FINSTART FINAL - TINGGAL TEMPEL

1. Tutup terminal backend dan frontend.
2. Backup folder C:\laragon\www\finstart untuk berjaga-jaga.
3. Ekstrak ISI ZIP ini.
4. Salin seluruh isi hasil ekstrak langsung ke C:\laragon\www\finstart.
5. Pilih "Replace the files in the destination".
6. Jangan hapus file lama berikut:
   - .git
   - backend\.env
   - frontend\.env (jika ada)
   - uploads / storage (jika ada)
7. Klik SELESAIKAN_TEMPEL.bat satu kali.
8. Klik JALANKAN_FINSTART.bat.

Paket ini tidak menjalankan seeder dan tidak mengganti akun/password yang sudah ada.
Seeder yang tersedia juga sudah diperbaiki agar tidak menimpa password user lama.

Jika login sudah telanjur rusak sebelum paket ini ditempel:
- Klik PERBAIKI_LOGIN.bat.
- Masukkan email akun yang ada di tabel users.
- Masukkan password baru minimal 8 karakter.
- Jalankan FinStart dan login kembali.

CATATAN:
- Session login disimpan di tabel auth_sessions. Jika tabel itu di-TRUNCATE, browser otomatis logout.
- Jangan menjalankan reset database yang menghapus auth_sessions bila ingin sesi tetap aktif.
- Login memakai kolom email, bukan nama pengguna.
